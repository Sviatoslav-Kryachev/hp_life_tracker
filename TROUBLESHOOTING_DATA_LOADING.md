# Решение проблемы с загрузкой данных

## Проблема
Данные не загружаются (показывается "Загрузка..." в карточках "Статистика по категориям" и "Мои цели")

## Возможные причины и решения

### 1. Сервер не запущен или запущен неправильно

**Решение:**
```bash
# Из корня проекта
cd Backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# Или используйте новый скрипт
python run_server.py
```

**Проверка:**
- Откройте http://localhost:8000 в браузере
- Должна открыться главная страница приложения

### 2. Проверьте консоль браузера

**Как проверить:**
1. Откройте DevTools (F12)
2. Перейдите на вкладку "Console"
3. Ищите ошибки (красные сообщения)

**Типичные ошибки:**
- `Failed to fetch` - сервер не запущен или недоступен
- `401 Unauthorized` - проблема с авторизацией
- `404 Not Found` - неправильный путь к API

### 3. Проверьте вкладку Network

**Как проверить:**
1. Откройте DevTools (F12)
2. Перейдите на вкладку "Network"
3. Обновите страницу
4. Ищите запросы к `/goals/` и `/xp/category-stats`
5. Проверьте статус ответа (должен быть 200)

### 4. Проверьте авторизацию

**Как проверить:**
1. Откройте DevTools (F12)
2. Перейдите на вкладку "Application" → "Local Storage"
3. Проверьте наличие ключа `token`
4. Если токена нет - войдите в систему заново

### 5. Проверьте пути к API

**В консоли браузера выполните:**
```javascript
console.log('API_BASE:', typeof API_BASE !== 'undefined' ? API_BASE : window.location.origin);
console.log('Token:', localStorage.getItem('token') ? 'Есть' : 'Нет');
```

### 6. Перезапустите сервер

После перемещения файлов в директории Frontend/Backend:
1. Остановите сервер (Ctrl+C)
2. Запустите заново из директории Backend
3. Очистите кэш браузера (Ctrl+Shift+R)

### 7. Проверьте логи сервера

При запуске сервера должны быть видны:
- Информация о запуске
- Ошибки (если есть)

## Быстрая диагностика

Выполните в консоли браузера:
```javascript
// Проверка функций
console.log('loadGoals:', typeof loadGoals);
console.log('loadCategoryStats:', typeof loadCategoryStats);

// Проверка API
fetch('/goals/', {
    headers: { "Authorization": `Bearer ${localStorage.getItem('token')}` }
}).then(r => r.json()).then(console.log).catch(console.error);
```

## Если ничего не помогло

1. Проверьте, что сервер запущен и доступен
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что вы авторизованы (есть токен)
4. Попробуйте перезапустить сервер
5. Очистите кэш браузера
